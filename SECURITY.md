# Security Policy

## Supported Versions

* ✅ = Recommended to use
* ❌ = Not recommended to use

| Version | Supported          |
| ------- | ------------------ |
| 1.5.x   | ✅                 |
| 1.4.0   | ✅                 |
| < 1.2.1 | :x:               |

## Reporting a Vulnerability
If you find a vulnerability or bug, please create an issue in Issues tab, we will look at it and fix it.
